## 实现功能
- 四则运算
- 括号表达式
- 小数输入
- 清屏和回退
- 取余
- 结果记录Ans键
- 三角函数（s c t分别对应sin cos tan）
- 科学计数法
- 阶乘
- 幂指数
- 用户输入错误处理
